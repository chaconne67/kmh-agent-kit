# 경쟁사 분석 (Competitive Intelligence)

한국 중소기업 B2B SaaS 시장에서의 경쟁 환경을 분석하고, 실전에서 즉시 활용 가능한 배틀카드를 생성하는 스킬입니다.

## 사용법

```
/sales competitors <대상 기업 URL 또는 이름>
/영업 경쟁사 <대상 기업 URL 또는 이름>
```

## 출력 파일

`COMPETITIVE-INTEL.md`

---

## 실행 절차

### 1단계: 경쟁사 탐지 (Competitor Detection)

대상 기업이 현재 사용 중인 솔루션을 다음 방법으로 탐지하십시오.

#### 1-1. 웹사이트 기술스택 분석
- WebFetch로 대상 기업 웹사이트를 조회하여 사용 중인 SaaS/IT 도구를 파악합니다.
- HTML 소스코드에서 스크립트 태그, 메타태그, 쿠키 등을 확인합니다.
- 예: Google Analytics, HubSpot, Salesforce, 채널톡, 카페24, NHN클라우드 등

#### 1-2. 채용공고 분석
- WebSearch로 `"[기업명]" 채용 site:jobkorea.co.kr OR site:saramin.co.kr OR site:wanted.co.kr`를 검색합니다.
- 채용 자격요건에 명시된 도구/플랫폼을 추출합니다.
- 예: "ERP 사용 경험", "세일즈포스 활용", "슬랙/노션 환경"

#### 1-3. 기업 프로필 조사
- 로켓펀치, 원티드 등에서 기업 프로필의 기술스택 섹션을 확인합니다.
- `"[기업명]" 기술스택 OR 사용도구 OR 협업도구`로 검색합니다.

#### 1-4. 경쟁사 후보 리스트업
- 탐지된 솔루션을 기반으로 직접 경쟁사(Direct Competitor)와 간접 경쟁사(Indirect Competitor)를 구분합니다.
- 카테고리별로 최대 5개 경쟁사를 리스트업합니다.

### 2단계: 한국 시장 평판 조사 (Korean Market Intelligence)

#### 2-1. 네이버 블로그/카페 리뷰
- WebSearch로 `"[경쟁사명]" 후기 OR 리뷰 site:blog.naver.com`을 검색합니다.
- WebSearch로 `"[경쟁사명]" 후기 OR 리뷰 site:cafe.naver.com`을 검색합니다.
- 실사용자 불만사항, 칭찬 포인트, 전환 사유를 정리합니다.

#### 2-2. 잡플래닛 기업 리뷰
- WebSearch로 `"[경쟁사명]" site:jobplanet.co.kr`을 검색합니다.
- 내부 직원 관점의 제품 품질, 고객 만족도 관련 언급을 수집합니다.

#### 2-3. IT 커뮤니티 평판
- 디스콰이엇, 요즘IT, GeekNews 등에서 해당 솔루션 관련 토론을 검색합니다.
- WebSearch로 `"[경쟁사명]" 장점 OR 단점 OR 비교`를 검색합니다.

#### 2-4. 뉴스/보도자료
- WebSearch로 `"[경쟁사명]" 투자 OR 인수 OR 파트너십`을 검색하여 최근 동향을 파악합니다.

### 3단계: 배틀카드 생성 (Battle Card per Competitor)

각 주요 경쟁사(최대 5개)에 대해 아래 구조의 배틀카드를 작성하십시오.

### 3-1. 배틀카드 작성 원칙

- **솔직함 우선**: 경쟁사의 강점을 과소평가하지 마십시오. 영업 현장에서 고객이 이미 알고 있는 사실을 부정하면 신뢰를 잃습니다.
- **검증 기반**: 약점은 반드시 리뷰, 뉴스, 사용자 피드백 등 출처가 있는 정보를 기반으로 작성하십시오.
- **실전 활용**: 지뢰 질문은 자연스러운 대화 흐름에서 사용할 수 있어야 합니다.

### 4단계: 기능 갭 분석 (Feature Gap Analysis)

주요 경쟁사와의 기능 비교표를 작성하십시오.

```markdown
## 기능 비교 매트릭스

| 기능/역량 | 우리 | 경쟁사A | 경쟁사B | 경쟁사C | 중요도 |
|-----------|------|---------|---------|---------|--------|
| [기능 1] | ✅/⚠️/❌ | ✅/⚠️/❌ | ✅/⚠️/❌ | ✅/⚠️/❌ | 상/중/하 |
| [기능 2] | | | | | |
| ... | | | | | |

✅ = 완전 지원 | ⚠️ = 부분 지원/제한적 | ❌ = 미지원
```

- **우리가 우세한 영역**: 고객 미팅에서 강조할 포인트
- **우리가 열세인 영역**: 사전에 답변을 준비해야 할 포인트
- **향후 로드맵 반영**: 곧 출시 예정인 기능은 별도 표시

### 6단계: 한국형 포지셔닝 전략 (Korean Positioning Strategy)

#### 한국 B2B 포지셔닝 원칙

1. **외교적 표현 (Diplomatic Positioning)**
   - 경쟁사를 직접 공격하지 마십시오.
   - ❌ "[경쟁사]는 기능이 부족합니다"
   - ✅ "저희는 [특정 영역]에 특화되어 있어서, 이 부분에서는 더 깊은 전문성을 제공해 드릴 수 있습니다"

2. **강점 인정 (Acknowledge Competitor Strengths)**
   - 고객이 경쟁사를 언급하면 강점을 먼저 인정하십시오.
   - ✅ "[경쟁사]도 좋은 솔루션이죠. 특히 [분야]에서는 잘 알려져 있습니다. 다만 저희가 차별화되는 부분은..."

3. **고객 중심 프레이밍 (Customer-Centric Framing)**
   - 경쟁사 비교가 아닌, 고객의 문제 해결에 초점을 맞추십시오.
   - ✅ "고객님의 상황에서는 [기능]이 특히 중요하실 텐데요, 이 부분에서 저희가..."

4. **관계 존중 (Respect Existing Relationships)**
   - 한국 비즈니스에서 기존 거래처 관계는 매우 중요합니다.
   - 기존 솔루션을 폄하하면 그 솔루션을 선택한 의사결정자의 체면을 깎는 것입니다.
   - ✅ "현재 사용하시는 솔루션으로도 잘 운영하고 계신 거죠. 저희는 보완적인 관점에서..."

## 출력 형식

최종 결과물은 `COMPETITIVE-INTEL.md` 파일로 저장하며, 아래 섹션을 포함합니다:

1. **요약 (Executive Summary)** — 3-5줄 핵심 요약
2. **경쟁 환경 개요 (Competitive Landscape)** — 경쟁사 맵
3. **배틀카드** — 경쟁사별 상세 배틀카드
4. **기능 갭 분석** — 비교 매트릭스
5. **승패 패턴** — 이기는/지는 시나리오
6. **한국형 포지셔닝 가이드** — 상황별 화법
7. **경쟁 전략 권고** — 실행 가능한 전략
