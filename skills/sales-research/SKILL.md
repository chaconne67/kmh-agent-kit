# 한국 기업 리서치 스킬 (Company Research Skill)

## 개요

이 스킬은 한국 중소기업(SMB) B2B SaaS 영업 대상 기업을 체계적으로 조사하여 COMPANY-RESEARCH.md 파일을 생성합니다. 한국 비즈니스 환경에 특화된 데이터 소스와 분석 프레임워크를 사용하십시오.

## 실행 절차

### 1단계: 웹사이트 크롤링 (Website Crawling)

대상 기업의 웹사이트에서 다음 페이지들을 WebFetch 도구로 수집하십시오. 각 페이지가 존재하지 않을 수 있으므로 오류 시 건너뛰십시오.

수집 대상 페이지:
- **메인 페이지**: `{domain}/`
- **회사 소개**: `{domain}/company/greeting`, `{domain}/about`, `{domain}/company/about`
- **제품/서비스**: `{domain}/product`, `{domain}/service`, `{domain}/solution`
- **채용 정보**: `{domain}/recruit`, `{domain}/careers`, `{domain}/jobs`
- **블로그/뉴스**: `{domain}/blog`, `{domain}/news`, `{domain}/notice`
- **IR/투자자 정보**: `{domain}/ir`, `{domain}/investor`

각 페이지에서 추출할 정보:
- 회사명, 대표이사명, 설립연도, 주소, 직원 수
- 주요 제품/서비스 설명
- 비전/미션 문구
- 고객사 로고 또는 목록
- 기술 스택 또는 기술 관련 키워드
- 채용 중인 포지션 (성장 지표)
- 최근 뉴스 또는 공지사항

### 2단계: 한국 공개 데이터 소스 검색 (Korean Public Data Sources)

WebSearch 도구를 사용하여 다음 소스들에서 기업 정보를 검색하십시오.

#### 필수 검색 소스
1. **DART 전자공시시스템** - 검색어: `"{기업명}" site:dart.fss.or.kr` 또는 `"{기업명}" DART 공시`
   - 재무제표, 사업보고서, 감사보고서 확인
2. **로켓펀치 (RocketPunch)** - 검색어: `"{기업명}" site:rocketpunch.com`
   - 기업 프로필, 직원 수, 투자 이력 확인
3. **잡코리아/사람인** - 검색어: `"{기업명}" site:jobkorea.co.kr` 또는 `"{기업명}" site:saramin.co.kr`
   - 채용 정보, 기업 리뷰, 연봉 정보 확인
4. **네이버 뉴스** - 검색어: `"{기업명}" site:news.naver.com`
   - 최근 6개월 뉴스 기사 확인
5. **공정거래위원회 기업집단 정보** - 검색어: `"{기업명}" 공정거래위원회` 또는 `"{기업명}" 기업 정보`

#### 보조 검색 소스
6. **크런치베이스 (Crunchbase)** - 검색어: `"{기업명}" site:crunchbase.com`
7. **블라인드** - 검색어: `"{기업명}" site:blind.co.kr` (기업 문화 파악)
8. **THE VC** - 검색어: `"{기업명}" site:thevc.kr` (스타트업 투자 정보)
9. **KED 한국기업데이터** - 검색어: `"{기업명}" 한국기업데이터`

#### 검색 팁
- 기업명은 반드시 큰따옴표로 감싸서 정확히 검색하십시오
- 영문 기업명과 한글 기업명을 모두 검색하십시오
- 대표이사명으로도 추가 검색하십시오: `"{대표이사명}" "{기업명}"`
- 최근 1년 이내 정보를 우선적으로 수집하십시오

### 3단계: 8개 차원 분석 (8-Dimension Analysis)

수집한 정보를 다음 8개 차원으로 분석하십시오.

#### 1. 기업개요 (Company Overview)
- 정식 회사명 (한글/영문)
- 설립연도, 법인 형태 (주식회사, 유한회사 등)
- 본사 소재지, 지사/지점
- 대표이사명
- 직원 수 (추정 포함)
- 업종 분류 (KSIC 한국표준산업분류 기준)

#### 2. 비즈니스 모델/매출 (Business Model / Revenue)
- 주요 수익원 (구독형, 라이선스형, 프로젝트형, 거래수수료형 등)
- 추정 연매출 규모 (가능한 경우)
- 주요 고객 세그먼트 (B2B, B2C, B2G)
- 매출 성장 추세 (공시 자료 기반)
- 수익 모델의 반복성 (Recurring Revenue 비율 추정)

#### 3. 제품/기술 (Products / Technology)
- 주요 제품/서비스 목록 및 설명
- 핵심 기술 또는 특허
- 기술 스택 (채용 공고 기반 추정)
- 제품 차별화 요소
- 최근 출시 제품 또는 업데이트

#### 4. 경영진 (Leadership)
- 대표이사 프로필 (학력, 경력, 인터뷰 발언)
- 주요 경영진 (CTO, CFO, COO 등)
- 경영진 변동 이력 (최근 1년)
- 이사회 구성 (상장사의 경우)

#### 5. 투자/재무 (Investment / Financials)
- 투자 유치 이력 (라운드, 금액, 투자자)
- 최근 재무 상태 (매출, 영업이익, 자산 규모)
- 정부 지원 사업 수혜 이력
- 신용등급 (가능한 경우)

#### 6. 시장 포지션 (Market Position)
- 속한 시장/산업의 규모 및 성장률
- 주요 경쟁사 목록
- 시장 점유율 추정
- 차별화 전략
- 산업 내 인지도/브랜드 파워

#### 7. 기업 문화 (Company Culture)
- 조직 문화 키워드 (블라인드, 잡코리아 리뷰 기반)
- 근무 환경 (재택근무, 유연근무 등)
- 의사결정 스타일 (수직적/수평적)
- 기술 도입 성향 (Early Adopter vs. Conservative)

#### 8. 최근 동향 (Recent Developments)
- 최근 6개월 주요 뉴스/이벤트
- 신규 사업 진출 또는 피벗
- 인수합병(M&A) 관련 소식
- 주요 파트너십 체결
- 조직 변동 (구조조정, 채용 확대 등)

### 4단계: Company Fit Score 산출 (0-100점)

다음 5개 항목을 각각 0-20점으로 평가하여 합산하십시오.

#### 기업 규모 적합성 (Size Fit) - 0~20점
| 조건 | 점수 |
|------|------|
| 직원 10-50명, 연매출 10-50억원 | 16-20점 |
| 직원 50-200명, 연매출 50-200억원 | 12-16점 |
| 직원 5-10명, 연매출 5-10억원 | 8-12점 |
| 직원 200명 이상 또는 5명 미만 | 0-8점 |

#### 산업 적합성 (Industry Fit) - 0~20점
| 조건 | 점수 |
|------|------|
| 우리 솔루션의 핵심 타겟 산업 | 16-20점 |
| 관련 산업, 유사 사용 사례 존재 | 12-16점 |
| 간접적 관련, 커스터마이징 필요 | 8-12점 |
| 무관한 산업 | 0-8점 |

#### 성장성 (Growth Potential) - 0~20점
| 조건 | 점수 |
|------|------|
| 최근 투자 유치, 채용 확대, 매출 성장 | 16-20점 |
| 안정적 성장, 신규 사업 진출 | 12-16점 |
| 정체 또는 소폭 성장 | 8-12점 |
| 축소, 구조조정, 재무 악화 | 0-8점 |

#### 기술 친화도 (Tech Readiness) - 0~20점
| 조건 | 점수 |
|------|------|
| SaaS/클라우드 이미 사용, IT 인력 보유 | 16-20점 |
| 디지털 전환 진행 중, 관심 높음 | 12-16점 |
| 기본적 IT 환경, 전환 의지 있음 | 8-12점 |
| IT 인프라 미비, 전환 의지 낮음 | 0-8점 |

#### 예산 여력 (Budget Capacity) - 0~20점
| 조건 | 점수 |
|------|------|
| 충분한 재무 여력, IT 투자 이력 있음 | 16-20점 |
| 정부 지원 사업 활용 가능 | 12-16점 |
| 제한적 예산, 소규모 투자 가능 | 8-12점 |
| 재무 상태 열악, 예산 확보 어려움 | 0-8점 |

#### 종합 등급
| 총점 | 등급 | 의미 |
|------|------|------|
| 80-100 | A등급 (최우선 공략) | 즉시 영업 활동 개시 |
| 60-79 | B등급 (적극 검토) | 추가 정보 수집 후 공략 |
| 40-59 | C등급 (관망) | 장기 육성 대상 |
| 0-39 | D등급 (보류) | 현재 시점 부적합 |
