# 이상적 고객 프로필 빌더 (Ideal Customer Profile Builder)

한국 중소기업(SMB) B2B SaaS 시장에 최적화된 이상적 고객 프로필(ICP)을 체계적으로 구축하는 스킬입니다.

## 사용법

```
/sales icp <제품/솔루션 설명 또는 URL>
/영업 ICP <제품/솔루션 설명 또는 URL>
```

## 출력 파일

`IDEAL-CUSTOMER-PROFILE.md` (300-400줄)

---

## 실행 절차

### 1단계: 솔루션 이해 (Solution Understanding)

먼저 우리 솔루션의 핵심 가치를 파악하십시오.

- 제품/서비스의 주요 기능
- 해결하는 핵심 문제
- 가격대 및 과금 모델
- 주요 성공 사례 (있는 경우)
- 한국 시장 타겟 (있는 경우)

#### 차원 5: 예산 적격성 (Budget Qualification)

```markdown
## 5. 예산 적격성 (Budget Qualification)

### 매출 기준 예산 추정
| 매출 규모 | 예상 IT 예산 (매출 대비) | 솔루션 예산 가용 범위 |
|-----------|------------------------|---------------------|
| 5-20억 | 1-2% (500-4,000만원) | 월 10-50만원 |
| 20-100억 | 1.5-3% (3,000만-3억) | 월 30-200만원 |
| 100-500억 | 2-4% (2-20억) | 월 100-500만원 |
| 500억 이상 | 2-5% (10억 이상) | 월 300만원 이상 |

### 정부지원사업 활용 가능성
| 지원사업 | 대상 | 지원 규모 | 시기 |
|---------|------|-----------|------|
| 클라우드 바우처 | 중소기업 | 최대 2,000만원 | 상반기 공고 |
| 데이터 바우처 | 중소·중견기업 | 최대 4,500만원 | 상반기 공고 |
| 스마트공장 지원 | 제조 중소기업 | 최대 1.5억원 | 연중 |
| DX바우처 | 중소기업 | 최대 3,000만원 | 상반기 공고 |
| K-디지털 플랫폼 | 중소기업 | 최대 5,000만원 | 수시 |
| 혁신바우처 | 중소기업 | 최대 1억원 | 연초 공고 |

> 정부지원사업을 활용하면 고객의 실질 부담이 크게 줄어듭니다.
> "정부지원사업 활용 가능 여부"는 한국 SMB 영업에서 매우 강력한 영업 앵글입니다.

### ROI 기대치
- 한국 SMB 의사결정자 기대 ROI: **투자 대비 3-5배** (6-12개월 내)
- 투자 회수 기간(Payback Period) 기대치: **6개월 이내** 선호
- ROI 증빙 방식: 레퍼런스 고객 사례 > 정량적 계산서 > 일반적 벤치마크
```

```markdown
## 네거티브 ICP (부적격 고객 기준)

> 아래 기준에 해당하는 기업은 영업 리소스를 투입하지 않는 것을 권고합니다.

### 즉시 제외 (Hard Disqualifiers)

### 주의 필요 (Soft Disqualifiers — 2개 이상 해당 시 재검토)
| 기준 | 설명 |
|------|------|

| 과거 3회 이상 미팅 후 미구매 | 정보 수집 목적일 가능성 |
```
