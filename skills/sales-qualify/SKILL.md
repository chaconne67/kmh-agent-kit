# 한국 SMB B2B 리드 검증 스킬 (Lead Qualification Skill)

## 개요

이 스킬은 한국 중소기업(SMB) B2B SaaS 영업에서 리드(잠재 고객)의 품질을 체계적으로 평가합니다. 한국형 BANT 분석, MEDDIC 분석, 긴급도(Urgency) 분석을 결합하여 종합 Opportunity Quality Score를 산출하고 LEAD-QUALIFICATION.md 파일을 생성합니다.

## 입력 정보

스킬 실행 전 다음 정보를 확인하십시오:
- 대상 기업명
- 기업 리서치 보고서 (COMPANY-RESEARCH.md가 있으면 참조)
- 접촉 이력 (있는 경우)
- 우리 솔루션 정보 (가격대, 타겟 고객 등)

## Part 1: 한국형 BANT 분석 (0-100점)

### B - Budget (예산) - 0~25점

#### 한국 SMB 매출 규모별 IT 예산 추정 기준

| 연매출 규모 | 추정 IT 예산 (연간) | SaaS 투자 가능액 | 기본 점수 |
|-------------|---------------------|-------------------|-----------|
| 5억원 미만 | 1,000-3,000만원 | 200-500만원 | 3-7점 |
| 5-20억원 | 3,000만-1억원 | 500-2,000만원 | 8-12점 |
| 20-50억원 | 1-3억원 | 2,000-5,000만원 | 13-17점 |
| 50-200억원 | 3-10억원 | 5,000만-2억원 | 18-22점 |
| 200억원 이상 | 10억원 이상 | 2억원 이상 | 23-25점 |

#### 예산 보너스/페널티 가점

| 조건 | 가감점 |
|------|--------|
| 정부 지원 사업 수혜 확인 (바우처, 스마트공장, 데이터바우처 등) | +3점 |
| 정부 지원 사업 신청 예정/검토 중 | +2점 |
| 최근 유사 SaaS 도입 이력 있음 | +2점 |
| 최근 투자 유치 완료 (6개월 이내) | +2점 |
| 재무 상태 악화 (영업손실 지속) | -3점 |
| 최근 구조조정/비용절감 중 | -3점 |
| 예산 집행 권한자 미확인 | -2점 |

#### 예산 확인 신호 (Budget Signals)

**긍정 신호:**
- "예산은 확보되어 있습니다"
- "올해 IT 투자 계획이 있습니다"
- "정부 바우처를 받았습니다"
- "견적을 요청드립니다"
- 유사 솔루션 이미 유료 사용 중

**부정 신호:**
- "무료 버전은 없나요?"
- "내년에 다시 검토하겠습니다"
- "예산이 빠듯합니다"
- "비용 대비 효과를 먼저 봐야 합니다"
- 무료 체험만 반복 요청

#### 정부 지원 사업 주요 목록 (참고용)

다음 정부 지원 사업은 고객의 예산 여력을 크게 확대시킵니다:
1. **중소기업 디지털전환(스마트제조) 지원사업** - 중소벤처기업부
2. **데이터바우처 지원사업** - 한국데이터산업진흥원
3. **AI 바우처 지원사업** - 정보통신산업진흥원
4. **클라우드 서비스 이용 지원사업** - 한국클라우드산업협회
5. **스마트공장 보급·확산사업** - 중소벤처기업부
6. **혁신바우처 사업** - 중소벤처기업부
7. **소상공인 디지털전환 지원사업** - 소상공인시장진흥공단
8. **ICT 이노베이션 지원사업** - 정보통신기획평가원

검색어: `"{기업명}" 정부지원사업`, `"{기업명}" 바우처`, `"{기업명}" 스마트공장`

### A - Authority (권한) - 0~25점

#### 한국 SMB 의사결정 구조

한국 SMB에서는 대표이사가 거의 모든 구매 의사결정을 직접 내립니다. 이는 서구 기업과 가장 큰 차이점입니다.

| 기업 규모 | 최종 의사결정자 | 영향력자 | 점수 기준 |
|-----------|----------------|----------|-----------|
| 1-10명 | 대표이사 (100%) | 없음 | 대표 접촉 시 25점 |
| 11-30명 | 대표이사 (90%) | 팀장/실장 | 대표 접촉 20-25점, 팀장 접촉 12-18점 |
| 31-100명 | 대표이사/이사 (70%) | 부서장/팀장 | 대표 접촉 18-25점, 부서장 접촉 12-18점 |
| 100-200명 | C-Level (50%) | 부서장, 구매팀 | C-Level 접촉 15-22점, 부서장 접촉 10-15점 |
| 200명 이상 | 부서장 이상 | 실무 담당자, 구매팀 | 부서장 이상 접촉 12-20점 |

#### 권한 확인 질문 (한국어 화법)

직접적으로 "결정권자가 누구시냐"고 묻는 것은 한국 비즈니스에서 실례가 될 수 있습니다. 다음과 같은 간접적 질문을 사용하십시오:

- "이런 종류의 도입 검토는 보통 어떤 절차로 진행되시나요?"
- "대표님께서도 이 부분에 관심을 가지고 계신가요?"
- "최종적으로 도입을 결정하실 때 어떤 분들이 함께 검토하시나요?"
- "이전에 비슷한 솔루션을 도입하셨을 때는 어떤 과정을 거치셨나요?"
- "검토 결과를 보고하실 분이 따로 계신가요?"

#### 권한 수준 판정

| 수준 | 설명 | 점수 |
|------|------|------|
| Level 1: 최종결정자 직접 접촉 | 대표이사 또는 최종결정자와 직접 대화 | 21-25점 |
| Level 2: 강력한 내부 지지자 | 결정자에게 직접 보고하는 임원/팀장이 적극 지지 | 16-20점 |
| Level 3: 중간 관리자 접촉 | 팀장급 접촉, 결정자 접근 경로 파악 | 11-15점 |
| Level 4: 실무자 접촉 | 담당자급만 접촉, 상위 보고 여부 불확실 | 6-10점 |
| Level 5: 접촉 없음 | 아직 유의미한 접촉 없음 | 0-5점 |

### N - Need (필요성) - 0~25점

#### 한국 SMB 디지털전환 니즈 신호

**강력한 니즈 신호 (Strong Need Signals) - 18-25점:**
- 디지털전환/DX 추진 공식 발표
- 스마트공장 도입 진행 중
- 기존 레거시 시스템 교체 검토 중
- 경쟁사 도입으로 인한 위기감 표명
- 정부 규제 변화에 대한 대응 필요 (전자세금계산서, 개인정보보호 등)
- 급격한 성장으로 인한 시스템 한계 경험
- RFP(제안요청서) 발행

**중간 니즈 신호 (Medium Need Signals) - 10-17점:**
- 관련 세미나/웨비나 참석 이력
- 경쟁사 솔루션 벤치마킹 중
- "이런 기능이 있으면 좋겠다" 언급
- 업무 효율화에 대한 관심 표명
- 유사 솔루션 무료 체험 이용 중
- 채용 공고에 관련 직무 등장

**약한 니즈 신호 (Weak Need Signals) - 3-9점:**
- 일반적인 정보 수집 단계
- "나중에 필요할 수도 있다" 수준
- 경쟁사가 사용한다는 이야기를 들었다는 정도
- 컨퍼런스에서 부스 방문

**니즈 없음 (No Need) - 0-2점:**
- 현재 시스템에 만족
- 관련 과제가 우선순위에 없음
- "지금은 괜찮습니다" 명시적 표현

#### 한국 특화 니즈 트리거

| 트리거 | 설명 | 가점 |
|--------|------|------|
| 정부 규제 대응 | 법률/규제 변화에 따른 필수 도입 | +5점 |
| 스마트공장/디지털전환 정책 | 정부 정책 방향과 일치 | +3점 |
| ESG 대응 | ESG 관련 시스템 필요 | +2점 |
| 수출 요건 | 해외 바이어 요구사항 충족 | +3점 |
| 세대교체 | 2세 경영인의 혁신 의지 | +3점 |

### T - Timeline (시기) - 0~25점

#### 한국 기업 예산 사이클 이해

한국 기업의 회계연도는 대부분 1월-12월입니다 (일부 3월-2월).

| 시기 | 특성 | 영업 전략 |
|------|------|-----------|
| **10-11월** | 차년도 예산 편성 | 예산에 포함시키기 위한 접근 최적기 |
| **1-3월 (Q1)** | 신규 예산 집행 시작, 정부 지원 사업 공고 | 신규 도입 최적기, 가장 높은 점수 |
| **4-6월 (Q2)** | 상반기 집행, 정부 사업 선정 | 양호한 시기 |
| **7-9월 (Q3)** | 하반기 시작, 예산 소진 압박 시작 | 잔여 예산 활용 기회 |
| **10-12월 (Q4)** | 잔여 예산 소진, 차년도 계획 | 연내 소진 또는 차년도 준비 |

#### 타임라인 점수 기준

| 도입 시기 | 점수 | 설명 |
|-----------|------|------|
| 즉시 (1개월 이내) | 22-25점 | 긴급한 니즈, 예산 확보 완료 |
| 단기 (1-3개월) | 17-21점 | 구체적 일정 논의 중 |
| 중기 (3-6개월) | 12-16점 | 검토 단계, 비교 평가 중 |
| 장기 (6-12개월) | 7-11점 | 차년도 계획에 포함 검토 |
| 미정 | 0-6점 | 구체적 시기 없음 |

#### 타임라인 가속 요인 (한국 특화)

| 요인 | 가점 |
|------|------|
| 정부 지원 사업 마감 임박 | +3점 |
| 연말 잔여 예산 소진 필요 | +2점 |
| 규제 시행일 확정 | +3점 |
| 경쟁사 도입 소식으로 긴급 검토 | +2점 |
| 대표이사 직접 지시로 검토 시작 | +3점 |

## Part 2: 한국형 MEDDIC 분석 (0-100%)

### M - Metrics (측정지표) - 판정: 확인됨/부분확인/미확인

고객이 측정 가능한 성과 목표를 가지고 있는지 평가합니다.

**확인 질문:**
- "이 프로젝트를 통해 구체적으로 어떤 수치를 개선하고 싶으신가요?"
- "현재 이 업무에 얼마나 시간/비용이 소요되고 계신가요?"
- "성공적인 도입의 기준이 무엇이라고 생각하시나요?"

**판정 기준:**
- **확인됨 (100%)**: 구체적 KPI 제시 (예: "처리 시간 50% 단축", "인건비 30% 절감")
- **부분확인 (50%)**: 방향성은 있으나 구체적 수치 없음 (예: "업무 효율화")
- **미확인 (0%)**: 측정 목표 없음 또는 확인 불가

### E - Economic Buyer (경제적 구매자) - 판정: 접촉완료/식별완료/미식별

최종 예산 승인권자를 식별하고 접촉했는지 평가합니다.

**한국 SMB 특화 기준:**
- 직원 50명 이하: **대표이사**가 경제적 구매자 (99% 확률)
- 직원 50-200명: **대표이사** 또는 **경영지원 담당 이사**
- 직원 200명 이상: **해당 부서 임원** 또는 **CFO**

**판정 기준:**
- **접촉완료 (100%)**: 경제적 구매자와 직접 대화 또는 미팅 완료
- **식별완료 (50%)**: 누구인지 파악했으나 아직 직접 접촉하지 못함
- **미식별 (0%)**: 경제적 구매자가 누구인지 파악하지 못함

### D - Decision Criteria (의사결정 기준) - 판정: 파악완료/부분파악/미파악

고객이 솔루션을 선택할 때 사용하는 기준을 파악했는지 평가합니다.

**한국 SMB의 일반적 의사결정 기준 (우선순위):**
1. 가격 (가성비)
2. 사용 편의성 (학습 비용 최소화)
3. 한국어 지원/한국 환경 최적화
4. 레퍼런스 (유사 업종 도입 사례)
5. 도입 후 지원 (기술지원, 교육)
6. 커스터마이징 가능 여부
7. 기존 시스템 연동 (ERP, 그룹웨어 등)
8. 데이터 보안/개인정보보호
9. 확장성
10. 브랜드 신뢰도

**판정 기준:**
- **파악완료 (100%)**: 고객의 평가 기준과 가중치를 구체적으로 파악
- **부분파악 (50%)**: 일부 기준만 파악, 가중치 불명확
- **미파악 (0%)**: 의사결정 기준 파악하지 못함

### D - Decision Process (의사결정 프로세스) - 판정: 파악완료/부분파악/미파악

구매 결정까지의 절차와 단계를 파악했는지 평가합니다.

**한국 SMB의 일반적 의사결정 프로세스:**
1. **니즈 인식** → 대표이사 또는 팀장이 필요성 인식
2. **정보 수집** → 담당자가 솔루션 조사, 데모 요청
3. **내부 검토** → 관련 팀 의견 수렴 (비공식적인 경우가 많음)
4. **비교 평가** → 2-3개 솔루션 비교 (가격표 위주)
5. **대표 보고** → 대표이사에게 추천안 보고
6. **최종 결정** → 대표이사 결재
7. **계약 체결** → 계약서 작성, 인감 날인

**한국 특화 주의사항:**
- 의사결정이 비공식 채널(식사 자리, 골프 등)에서 이루어지는 경우가 많습니다
- 급한 경우 대표이사 판단으로 당일 결정도 가능합니다
- 반대로 "검토 중"이 수개월 지속되는 경우도 흔합니다
- 연말/분기말에 결정이 집중되는 경향이 있습니다

**판정 기준:**
- **파악완료 (100%)**: 프로세스 단계, 소요 시간, 관련자를 구체적으로 파악
- **부분파악 (50%)**: 대략적 절차만 파악
- **미파악 (0%)**: 프로세스 파악하지 못함

### I - Identify Pain (핵심 과제 식별) - 판정: 명확/암시적/미식별

고객이 해결하고자 하는 핵심 과제(Pain Point)를 식별했는지 평가합니다.

**한국 SMB 공통 과제 목록:**

| 카테고리 | 구체적 과제 |
|----------|------------|
| 업무 효율 | 수작업 과다, 엑셀 의존, 중복 입력, 정보 단절 |
| 인력 | 인력 부족, 높은 이직률, 교육 비용, 핵심인력 의존 |
| 성장 | 시스템 한계로 성장 정체, 수동 프로세스의 확장 불가 |
| 규제 | 개인정보보호, 세무, 노무 관련 컴플라이언스 |
| 경쟁 | 경쟁사 대비 디지털 격차, 고객 경험 차별화 어려움 |
| 데이터 | 데이터 기반 의사결정 불가, 현황 파악 지연, 보고서 수동 작성 |
| 비용 | 불필요한 비용 발생, 원가 관리 어려움, ROI 측정 불가 |

**확인 질문 (한국식 간접 화법):**
- "지금 가장 시간이 많이 걸리시는 업무가 어떤 건가요?"
- "최근에 이 부분 때문에 불편하셨던 경험이 있으신가요?"
- "대표님께서 특별히 개선을 강조하시는 영역이 있으신가요?"
- "만약 이 문제가 해결되면 가장 크게 달라지는 게 무엇일까요?"

**판정 기준:**
- **명확 (100%)**: 고객이 직접 과제를 명시하고 해결 의지를 표현
- **암시적 (50%)**: 과제가 존재할 것으로 추정되나 고객이 명시적으로 인정하지 않음
- **미식별 (0%)**: 핵심 과제를 파악하지 못함

### C - Champion (내부 지지자) - 판정: 확보/후보/없음

우리 솔루션을 내부에서 적극적으로 추진해줄 사람을 확보했는지 평가합니다.

**한국 SMB에서의 Champion 특성:**
- SMB에서는 Champion = 대표이사인 경우가 가장 이상적입니다
- 팀장급이 Champion인 경우, 대표에게 보고하고 설득하는 역할을 합니다
- Champion이 실무 담당자(대리/사원급)인 경우 의사결정 영향력이 제한적입니다

**이상적인 Champion 조건:**
- 대표이사에게 직접 보고할 수 있는 위치
- 현재 과제를 직접 겪고 있어 해결 의지가 강함
- 사내에서 신뢰와 영향력을 가진 인물
- 우리 솔루션에 대해 긍정적이고 적극적
- 내부 회의에서 우리를 대변할 의지가 있음

**Champion 확인 질문:**
- "이 프로젝트를 사내에서 주도적으로 추진하시는 분이 계신가요?"
- "대표님께 보고하실 때 저희가 도움을 드릴 수 있는 부분이 있을까요?"
- "내부적으로 이 방향에 대해 지지하시는 분이 계신가요?"

**판정 기준:**
- **확보 (100%)**: Champion이 명확하고, 적극적으로 내부 추진 중
- **후보 (50%)**: 가능성 있는 인물을 식별했으나 아직 적극적 지지 확인 안 됨
- **없음 (0%)**: Champion이 없거나 식별하지 못함

## Part 5: Opportunity Quality Score 산출

### 종합 점수 산출 공식

```
Opportunity Quality Score = BANT점수 × 50% + MEDDIC점수 × 30% + 긴급도점수 × 20%
```

#### BANT 점수 (0-100)
- Budget(0-25) + Authority(0-25) + Need(0-25) + Timeline(0-25) = 0-100

#### MEDDIC 점수 (0-100)
각 항목의 % 값을 합산하여 평균:
- (Metrics + Economic Buyer + Decision Criteria + Decision Process + Identify Pain + Champion) / 6 × 100

#### 긴급도 점수 (0-100)
- 시간압박(0-25) + 과제심각도(0-25) + 경쟁압박(0-25) + 내부모멘텀(0-25) = 0-100

### 종합 등급 판정

| 점수 | 등급 | 한국어 라벨 | 영업 조치 |
|------|------|-------------|-----------|
| 75-100 | **A등급** | **즉시 공략 (Hot Lead)** | 48시간 내 미팅 설정, 전담 영업 배정, 주간 팔로업 |
| 50-74 | **B등급** | **적극 추진 (Warm Lead)** | 1주 내 접촉, 추가 정보 수집, 격주 팔로업 |
| 25-49 | **C등급** | **육성 대상 (Nurturing Lead)** | 월간 컨텐츠 전송, 이벤트 초대, 분기 체크인 |
| 0-24 | **D등급** | **보류 (Cold Lead)** | CRM에 기록, 6개월 후 재검토, 자동 뉴스레터만 |
