# 한국 SMB 의사결정자 맵핑 스킬 (Decision Maker Mapping Skill)

## 개요

이 스킬은 한국 중소기업(SMB) B2B SaaS 영업에서 대상 기업의 의사결정자를 체계적으로 식별하고, 접근 전략을 수립하여 DECISION-MAKERS.md 파일을 생성합니다. 한국 SMB의 특수한 의사결정 구조와 직급 체계를 반영합니다.

## 핵심 원칙: 한국 SMB 의사결정 구조

### 가장 중요한 사실

**한국 SMB(직원 50명 이하)에서는 대표이사가 거의 모든 구매 의사결정을 직접 내립니다.**

이것은 서구 B2B 영업과의 가장 큰 차이점입니다. 서구에서는 부서장, 구매팀, 위원회 등이 의사결정에 관여하지만, 한국 SMB에서는 대표이사 1인의 판단이 절대적입니다.

따라서:
- 대표이사에게 도달하는 것이 최우선 목표입니다
- 중간 관리자를 통한 접근은 차선책입니다
- 대표이사를 만나지 못하면 영업 성공 확률이 크게 낮아집니다

## 2단계: 의사결정자 검색 (Korean Sources)

다음 플랫폼을 우선순위 순으로 검색하십시오.

### 검색 소스 우선순위

#### 1순위: 로켓펀치 (RocketPunch)
- **검색어**: `"{기업명}" site:rocketpunch.com`
- **확인 항목**: 대표이사명, 경영진, 직원 목록, 직급
- **장점**: 한국 스타트업/IT기업 정보 가장 풍부
- **프로필 정보**: 학력, 경력, 소개

#### 2순위: 원티드 (Wanted)
- **검색어**: `"{기업명}" site:wanted.co.kr`
- **확인 항목**: 채용 담당자, 팀 구성, 직무 정보
- **장점**: IT/스타트업 인재 플랫폼, 팀 구조 파악 가능

#### 3순위: 링크드인 (LinkedIn)
- **검색어**: `"{기업명}" site:linkedin.com`
- **확인 항목**: 경영진 프로필, 경력, 인맥
- **장점**: 글로벌 네트워크, 경력 이력 상세
- **주의**: 한국 SMB 대표들의 링크드인 활용률은 상대적으로 낮습니다

#### 4순위: 잡코리아/사람인
- **검색어**: `"{기업명}" 채용 site:jobkorea.co.kr`
- **확인 항목**: 인사담당자, 채용 부서, 조직 구조
- **장점**: 채용 공고를 통한 조직 구조 파악

#### 추가 소스
- **DART 전자공시**: 상장사/외감법인의 이사회 구성, 등기이사 확인
- **네이버 인물검색**: `"{대표이사명}" "{기업명}"`
- **구글 이미지 검색**: 대표이사 얼굴, 행사 사진
- **기업 블로그/뉴스룸**: 경영진 인터뷰, 조직 소개
- **유튜브**: 대표이사 강연, 인터뷰 영상

## 3단계: 구매 위원회 맵핑 (Buying Committee)

### 한국 SMB 간소화된 구매 위원회

한국 SMB에서는 서구의 복잡한 구매 위원회(6-10명) 대신 일반적으로 3-5명이 관여합니다.

#### 핵심 역할 3가지

| 역할 | 한국 SMB에서의 해당자 | 영향력 | 접근 전략 |
|------|----------------------|--------|-----------|
| **Economic Buyer (경제적 구매자)** | **대표이사** | 최종 결정 100% | 직접 접촉 최우선, ROI 중심 대화 |
| **Champion (내부 지지자)** | **팀장/실장급** | 내부 추천 및 추진 | 실무적 가치 입증, 내부 설득 자료 제공 |
| **Technical Evaluator (기술 평가자)** | **IT 담당자/개발팀장** | 기술 적합성 평가 | 기술 스펙, 연동성, 보안 중심 대화 |

#### 보조 역할 (기업 규모에 따라)

| 역할 | 한국 SMB에서의 해당자 | 영향력 | 접근 전략 |
|------|----------------------|--------|-----------|
| **End User (최종 사용자)** | 실무 담당자 (대리/사원) | 사용성 피드백 | 사용 편의성 데모, 교육 지원 |
| **Gatekeeper (문지기)** | 비서/경영지원팀 | 접근 차단/허용 | 정중한 관계 구축, 가치 간결히 전달 |
| **Influencer (영향력자)** | 외부 자문, 거래처 대표 | 간접 영향 | 네트워크 활용, 추천 확보 |

### 기업 규모별 구매 위원회 구성

#### 직원 1-10명 (마이크로 기업)
```
의사결정: 대표이사 (단독 결정)
영향력자: 없음 또는 공동 창업자
실무: 대표이사가 직접 검토
```
→ **전략**: 대표이사 1인에게 집중하십시오

#### 직원 11-30명 (소기업)
```
의사결정: 대표이사 (최종 결정)
추천자: 팀장 1명
실무 검토: 담당자 1명
```
→ **전략**: 팀장을 Champion으로 확보하고, 대표 미팅을 요청하십시오

#### 직원 31-100명 (중소기업)
```
의사결정: 대표이사 (최종 승인)
검토/추천: 해당 부서장/이사
기술 평가: IT 담당 또는 개발팀장
실무 검토: 실무 담당자 1-2명
```
→ **전략**: 부서장을 통해 진입하고, Champion으로 육성 후 대표 보고에 동행하십시오

#### 직원 100-200명 (중기업)
```
의사결정: 대표이사/경영진 회의
검토/추천: 담당 임원(이사급)
기술 평가: IT팀/개발팀
실무 검토: 2-3명
구매 절차: 경영지원팀/총무팀
```
→ **전략**: 담당 임원 접촉 후 경영진 보고 자료를 함께 준비하십시오

## 4단계: 개인화 앵커 수집 (Personalization Anchors)

각 의사결정자에 대해 다음 개인화 앵커를 수집하십시오. 이 정보는 이후 아웃리치 및 미팅 준비에 활용됩니다.

### 수집 대상 정보

#### 1. 인터뷰/기고문
- **검색어**: `"{인물명}" 인터뷰`, `"{인물명}" 기고`
- **활용**: 경영 철학, 관심사, 고민 파악
- **대화 활용 예**: "대표님께서 OO 인터뷰에서 말씀하신 OO에 공감하여 연락드립니다"

#### 2. 블로그/SNS 활동
- **검색 대상**: 네이버 블로그, 브런치, 페이스북, 인스타그램
- **검색어**: `"{인물명}" "{기업명}" 블로그`, `"{인물명}" 브런치`
- **활용**: 관심 분야, 가치관, 일상적 소통 스타일 파악

#### 3. 컨퍼런스/세미나 발표
- **검색어**: `"{인물명}" 발표`, `"{인물명}" 컨퍼런스`, `"{인물명}" 세미나`
- **활용**: 전문 분야, 업계 인지도, 최근 관심 주제 파악
- **대화 활용 예**: "OO 컨퍼런스에서 발표하신 내용을 인상 깊게 들었습니다"

#### 4. 학력/동문 네트워크
- **검색어**: 로켓펀치/링크드인 프로필 확인
- **활용**: 학연(학교 동문) 기반 Warm Introduction 가능성
- **주요 확인**: 대학교, 대학원, MBA, 해외 유학 여부

#### 5. 수상/선정 이력
- **검색어**: `"{인물명}" 수상`, `"{기업명}" 수상`, `"{인물명}" 선정`
- **활용**: 축하 메시지를 통한 자연스러운 접근
- **대화 활용 예**: "최근 OO 수상을 축하드립니다. 관련하여..."

#### 6. 저서/출판물
- **검색어**: `"{인물명}" 저자`, `"{인물명}" 책`
- **활용**: 깊은 관심사 파악, 전문성 인정
- **대화 활용 예**: "대표님 저서 OO을 읽고 감명받았습니다"

## 5단계: Warm Path 탐색 (따뜻한 접근 경로)

한국 B2B 영업에서 콜드 접근보다 소개/추천을 통한 접근이 훨씬 효과적입니다. 다음 Warm Path를 탐색하십시오.

### Warm Path 유형 및 우선순위

#### 1. 학연 (Alumni Network) - 최우선
- **확인 방법**: 로켓펀치/링크드인에서 학력 확인
- **활용**: 같은 대학 동문, 같은 학과, 같은 동아리
- **접근법**: "OO대학교 선배/후배/동문으로서 인사드립니다"
- **효과**: 한국에서 학연은 매우 강력한 신뢰 기반입니다

#### 2. 업종 협회/단체 (Industry Association)
- **확인 방법**: `"{기업명}" 협회`, `"{기업명}" 회원사`
- **주요 협회**: 한국소프트웨어산업협회, 한국인터넷기업협회, 대한상공회의소, 중소기업중앙회 등
- **활용**: 같은 협회 회원, 협회 행사에서의 만남
- **접근법**: "OO 협회 행사에서 뵙게 되어 연락드립니다"

#### 3. 공통 투자자/멘토 (Shared Investors/Mentors)
- **확인 방법**: THE VC, 로켓펀치에서 투자자 확인
- **활용**: 같은 VC 포트폴리오, 같은 액셀러레이터 출신
- **접근법**: 투자자/멘토를 통한 소개 요청

#### 4. 지역 네트워크 (Regional Network)
- **확인 방법**: 본사 소재지 확인
- **활용**: 같은 지역 기업인 모임, 테크노파크, 창조경제혁신센터
- **주요 지역 네트워크**: 판교/성남, 강남/서초, 구로/금천, 대전, 부산, 대구 등
- **접근법**: "같은 OO 지역에서 사업하시는 인연으로..."

#### 5. 기존 고객 추천 (Customer Referral)
- **확인 방법**: 기존 고객 중 대상 기업과 관계 있는 곳 탐색
- **활용**: 기존 만족 고객이 직접 소개
- **접근법**: "저희 고객사인 OO 대표님께서 소개해 주셔서 연락드립니다"
- **효과**: 가장 강력한 Warm Path입니다

#### 6. 행사/컨퍼런스 (Events)
- **확인 방법**: 대상 인물의 행사 참석/발표 이력
- **활용**: 같은 행사 참석자로서 자연스러운 접근
- **주요 행사**: COMEUP, 넥스트라이즈, beLAUNCH, 코리아핀테크위크 등

## 6단계: Contact Access Score 산출 (0-100점)

### 평가 항목

| 항목 | 배점 | 설명 |
|------|------|------|
| 의사결정자 식별 (Identification) | 0-20점 | 핵심 의사결정자를 얼마나 명확히 식별했는가 |
| 연락처 확보 (Contact Info) | 0-20점 | 전화번호, 이메일 등 직접 연락 수단 확보 여부 |
| Warm Path 존재 (Warm Path) | 0-20점 | 소개/추천 경로가 있는가 |
| 개인화 정보 (Personalization) | 0-20점 | 대화에 활용할 개인화 앵커가 있는가 |
| 구매 위원회 파악 (Buying Committee) | 0-20점 | 의사결정 구조와 관련자를 파악했는가 |

### 세부 채점 기준

#### 의사결정자 식별 (0-20점)
| 수준 | 점수 |
|------|------|
| 대표이사 + 모든 관련자 식별 완료 | 18-20점 |
| 대표이사 식별, 일부 관련자 파악 | 14-17점 |
| 대표이사명만 확인 | 10-13점 |
| 담당 부서/직급만 추정 | 5-9점 |
| 식별 불가 | 0-4점 |

#### 연락처 확보 (0-20점)
| 수준 | 점수 |
|------|------|
| 대표이사 직접 연락처 (휴대폰/이메일) 확보 | 18-20점 |
| 대표이사 회사 연락처 + 관련자 연락처 | 14-17점 |
