# 한국 비즈니스 미팅 준비 스킬 (Meeting Preparation Skill)

## 개요

이 스킬은 한국 SMB B2B SaaS 영업 미팅을 체계적으로 준비합니다. 11개 섹션의 종합 미팅 준비 자료를 생성하고, 한국 비즈니스 예절과 문화적 뉘앙스를 반영하여 MEETING-PREP.md 파일을 작성합니다.

## 입력 정보

미팅 준비에 필요한 정보를 확인하십시오:
- 미팅 대상 기업명
- 미팅 참석자 (이름, 직급)
- 미팅 일시 및 장소 (온라인/오프라인)
- 미팅 목적 (첫 미팅/데모/제안/협상 등)
- 미팅 시간 (30분/60분)
- 기존 리서치 파일 (COMPANY-RESEARCH.md, DECISION-MAKERS.md 등 참조)

## 섹션 6: 대화 포인트 5-7개 (Talking Points)

미팅에서 전달해야 할 핵심 메시지를 5-7개로 준비하십시오.

### 대화 포인트 구성 원칙

1. **기업 맥락과 연결**: 일반적 제품 설명이 아닌 {기업명}의 상황에 맞춘 메시지
2. **고객 언어 사용**: 우리 용어가 아닌 고객이 사용하는 용어
3. **가치 중심**: 기능(Feature) 나열이 아닌 가치(Value/Benefit) 전달
4. **증거 기반**: 주장에 근거(사례, 수치) 뒷받침

### 대화 포인트 템플릿

```
[대화 포인트 1: {제목}]
핵심 메시지: {한 줄 메시지}
근거/사례: {뒷받침 자료}
전환 화법: "{자연스러운 전환 멘트}"

[대화 포인트 2: {제목}]
...

{5-7개 작성}
```

### 대화 포인트 순서 가이드

| 순서 | 유형 | 예시 |
|------|------|------|
| 1 | 공감/인정 | "최근 {성과}를 보면서 인상적이었습니다" |
| 2 | 과제 공감 | "{업종}에서 {과제}가 공통적인 고민이시죠" |
| 3 | 접근 방법 | "저희는 이런 과제를 {접근법}으로 해결합니다" |
| 4 | 차별적 가치 | "특히 {차별점}이 {기업명}에 맞을 것 같습니다" |
| 5 | 사회적 증거 | "{유사 기업}에서 {성과}를 달성했습니다" |
| 6 | ROI/효과 | "예상되는 효과는 {수치}입니다" |
| 7 | 다음 단계 | "다음 단계로 {제안}을 제안드립니다" |

## 섹션 7: 탐색 질문 10개 (Discovery Questions)

SPIN(Situation-Problem-Implication-Need-Payoff) 프레임워크를 한국식 간접 화법으로 적용합니다.

### SPIN 질문 한국어 적용

#### Situation (상황 질문) - 3개
현재 상황을 파악하되, 이미 알고 있는 정보를 묻지 않습니다.

```
1. "현재 {업무 영역}은 어떤 방식으로 관리하고 계신가요?"
   → 목적: 현재 프로세스/도구 파악
   → 팔로업: "그 과정에서 어려우신 점은 없으신지요?"

2. "팀 내에서 {업무}를 담당하시는 분이 몇 분 정도 되시나요?"
   → 목적: 조직 규모/구조 파악
   → 팔로업: "인원 대비 업무량은 어떠신가요?"

3. "현재 사용하고 계신 시스템이나 도구가 있으신가요?"
   → 목적: 기존 솔루션 파악
   → 팔로업: "그 도구에서 가장 만족하시는 부분은 어떤 건가요?"
```

#### Problem (문제 질문) - 3개
과제를 인식하게 하되, 직접적으로 "문제가 있다"고 하지 않습니다.

```
4. "혹시 {업무 영역}에서 시간이 많이 걸리시거나 불편하신 부분이 있으신가요?"
   → 목적: 과제 표면화
   → 주의: 상대방이 스스로 말하게 유도, 단정짓지 않기

5. "대표님께서 특별히 개선을 원하시는 영역이 있으실까요?"
   → 목적: 우선순위 파악
   → 주의: 대표이사의 관심사 = 구매 결정 핵심

6. "비슷한 업종의 다른 기업분들은 {과제}를 많이 언급하시는데, {기업명}에서도 비슷한 경험이 있으신지요?"
   → 목적: 간접적 과제 제시
   → 주의: 다른 기업 사례를 통한 자연스러운 유도
```

#### Implication (시사 질문) - 2개
과제를 방치할 경우의 영향을 인식하게 합니다.

```
7. "지금 말씀하신 부분이 해결되지 않으면, 향후 {기간} 뒤에는 어떤 상황이 될 것 같으신가요?"
   → 목적: 과제의 심각성 인식
   → 주의: 과도한 공포 유발은 역효과

8. "이 부분이 매출이나 고객 만족에도 영향을 미치고 있진 않으신지요?"
   → 목적: 비즈니스 임팩트 연결
   → 주의: 부드럽게 질문, 상대방의 판단 존중
```

#### Need-Payoff (가치 질문) - 2개
해결 시의 가치를 상대방이 직접 말하게 합니다.

```
9. "만약 {과제}가 해결된다면, 가장 먼저 달라질 부분이 무엇이라고 생각하시나요?"
   → 목적: 고객이 직접 가치를 표현하게 유도
   → 후속: 고객의 말을 인용하여 솔루션 연결

10. "{구체적 효과}가 가능하다면, {기업명}에 어떤 의미가 있을까요?"
    → 목적: ROI를 고객 관점에서 확인
    → 후속: 구체적 수치로 뒷받침
```

### 질문 규칙

- **경청 비율**: 70% 듣기, 30% 말하기를 유지하십시오
- **메모**: 고객의 표현을 그대로 메모하십시오 (나중에 제안서에 활용)
- **속도**: 질문을 연달아 하지 말고, 충분한 답변 시간을 주십시오
- **공감**: 답변 후 반드시 공감/인정 표현을 하십시오 ("네, 충분히 이해합니다")
- **유연성**: 고객 답변에 따라 준비된 질문 순서를 유연하게 조정하십시오

## 섹션 8: 예상 이의 5개 (Anticipated Objections)

### 이의 처리 프레임워크: 경청-공감-질문-제안 (LEQP)

| 단계 | 한국어 | 설명 | 예시 표현 |
|------|--------|------|-----------|
| **L** (Listen) | **경청** | 끝까지 듣고 메모 | (끄덕이며 경청) |
| **E** (Empathize) | **공감** | 상대의 우려를 인정 | "네, 그 부분은 당연히 걱정되실 수 있습니다." |
| **Q** (Question) | **질문** | 이의의 진짜 이유를 파악 | "혹시 구체적으로 어떤 부분이 가장 걱정되시나요?" |
| **P** (Propose) | **제안** | 해결 방안 제시 | "저희가 이런 방식으로 도와드릴 수 있습니다." |

### 한국 SMB 공통 이의 5가지

#### 이의 1: "가격이 부담됩니다"
```
[경청] (가격 우려를 끝까지 경청)
[공감] "네, 비용 부분은 당연히 신중하게 검토하셔야 합니다."
[질문] "혹시 예산 범위를 대략 말씀해 주실 수 있으신가요?
        맞춤형으로 방안을 찾아보겠습니다."
[제안] 옵션 A: "단계적 도입으로 초기 비용을 낮추실 수 있습니다."
       옵션 B: "정부 지원 사업(바우처)을 활용하시면 실질 부담이 {N}%까지 줄어듭니다."
       옵션 C: "{유사 기업}에서는 도입 {기간} 후 투자비를 회수하셨습니다."
```

#### 이의 2: "지금은 시기가 아닌 것 같습니다"
```
[경청] (시기에 대한 의견 경청)
[공감] "네, 시기 선택이 중요하시죠. 충분히 이해합니다."
[질문] "혹시 특별히 기다리고 계신 시점이나 조건이 있으신가요?"
[제안] 옵션 A: "그러시다면 {시점}에 다시 말씀드려도 될까요?"
       옵션 B: "지금 미리 검토해 두시면 {예산편성기/정부사업}에 맞춰 진행하실 수 있습니다."
       옵션 C: "무료 체험부터 가볍게 시작해 보시는 건 어떠실까요?"
```

#### 이의 3: "내부적으로 검토가 더 필요합니다"
```
[경청] (내부 프로세스 존중)
[공감] "물론입니다. 신중하게 검토하시는 것이 좋습니다."
[질문] "내부 검토에 필요한 추가 자료가 있으시면 저희가 준비해 드리겠습니다.
        혹시 어떤 부분을 중점적으로 검토하실 예정이신가요?"
[제안] 옵션 A: "내부 보고용 요약 자료를 만들어 드릴까요?"
       옵션 B: "대표님께 직접 설명드릴 기회를 주시면 감사하겠습니다."
       옵션 C: "검토 결과를 {날짜}쯤 확인차 연락드려도 될까요?"
```

## 섹션 10: 한국 비즈니스 예절 (Korean Business Etiquette)

### 10-1. 명함 교환 (Business Card Exchange)

| 단계 | 행동 | 주의사항 |
|------|------|----------|
| 1 | 일어서서 양손으로 명함을 건넵니다 | 한 손으로 건네지 마십시오 |
| 2 | 상대의 직급이 높으면 약간 고개를 숙이며 건넵니다 | 명함의 글씨가 상대 쪽을 향하게 |
| 3 | 양손으로 받으면서 명함을 확인합니다 | 바로 주머니에 넣지 마십시오 |
| 4 | "OO 대표님이시군요" 등 확인 멘트 | 이름과 직함을 소리내어 확인 |
| 5 | 테이블 위에 직급 순서대로 놓습니다 | 미팅 중 참고, 위에 물건 올리지 않기 |
| 6 | 미팅 종료 후 명함 케이스에 정리 | 접거나 메모하지 마십시오 |

### 10-2. 좌석 배치 (Seating Arrangement)

```
[회의실 일반 배치]

         문(Door)
         ↓
┌──────────────────────┐
│                      │
│  [하석4] [하석3]     │  ← 문에 가까운 쪽 = 하석 (아랫자리)
│          테이블      │
│  [상석2] [상석1]     │  ← 문에서 먼 쪽 = 상석 (윗자리)
│                      │
└──────────────────────┘
```

- **상석 (윗자리)**: 문에서 가장 먼 자리, 최고 직급자
- **하석 (아랫자리)**: 문에서 가장 가까운 자리, 최하 직급자
- **방문자 상석**: 손님(고객) 측 최고 직급자가 상석
- **실무 원칙**: 고객이 먼저 자리를 잡도록 안내하십시오

### 10-3. 인사 순서 (Greeting Order)

| 순서 | 규칙 | 예시 |
|------|------|------|
| 1 | 최고 직급자끼리 먼저 인사 | 우리 측 대표 → 상대 대표 |
| 2 | 방문자(영업)가 먼저 명함을 건넴 | "먼저 인사드리겠습니다" |
| 3 | 직급 순서대로 소개 | 높은 직급부터 낮은 직급 순 |
| 4 | 악수 시 가벼운 허리 인사 동반 | 과도하게 숙이지 않기 |
| 5 | "잘 부탁드리겠습니다"로 마무리 | |

### 10-4. 식사 예절 (Dining Etiquette)

미팅 후 식사로 이어지는 경우:

| 상황 | 예절 |
|------|------|
| 메뉴 선택 | 상대(고객)에게 먼저 권합니다 "뭐 드시고 싶으신 거 있으세요?" |
| 음식 시작 | 최고 직급자가 먼저 수저를 들 때까지 기다립니다 |
| 술자리 | 상대에게 따라드릴 때 두 손으로, 받을 때도 두 손 |
| 건배 | 상대보다 잔을 낮추어 부딪칩니다 |
| 계산 | 초대한 쪽(영업 측)이 자연스럽게 계산합니다 |
| 첫 식사 | 부담스럽지 않은 수준의 식사를 선택합니다 |

### 10-5. 온라인 미팅 예절

| 항목 | 기준 |
|------|------|
| 카메라 | 반드시 ON (상대방도 ON 요청) |
| 배경 | 정돈된 배경 또는 가상 배경 |
| 복장 | 오프라인과 동일하게 비즈니스 캐주얼 이상 |
| 시작 | 1-2분 전 입장, 상대보다 먼저 대기 |
| 음소거 | 상대 발언 시 음소거, 내 발언 시 해제 |
| 화면 공유 | 불필요한 탭/알림 미리 정리 |
| 마무리 | 상대가 먼저 퇴장할 수 있도록 잠시 대기 |
