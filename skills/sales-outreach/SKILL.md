# 한국형 멀티채널 아웃리치 시퀀스 스킬 (Outreach Sequence Skill)

## 개요

이 스킬은 한국 중소기업(SMB) B2B SaaS 영업에서 잠재 고객에게 처음 접촉하는 아웃리치 시퀀스를 설계합니다. 서구식 콜드 이메일 접근법을 완전히 재구성하여 한국 비즈니스 문화에 맞는 멀티채널 시퀀스를 생성하고 OUTREACH-SEQUENCE.md 파일을 작성합니다.

## 핵심 원칙: 한국형 아웃리치의 차이점

### 서구 vs 한국 접근법 비교

| 항목 | 서구 (Western) | 한국 (Korean) |
|------|----------------|---------------|
| **1순위 채널** | 이메일 | 소개/추천 |
| **2순위 채널** | 링크드인 | 전화 |
| **3순위 채널** | 전화 | 카카오톡 비즈 메시지 |
| **어조** | 직접적, 가치 제안 선도 | 간접적, 관계 구축 선도 |
| **존칭** | 이름 (first name) | 직함 + 존댓말 (OO 대표님) |
| **메시지 길이** | 짧고 직접적 | 정중하고 맥락 제공 |
| **접근 방식** | 문제 → 솔루션 제시 | 관계 → 이해 → 제안 |
| **성공 핵심** | 가치 제안 명확성 | 신뢰와 관계 |

### 절대 하지 말아야 할 것
- 처음부터 가격이나 할인을 언급하지 마십시오
- 반말이나 이름만으로 호칭하지 마십시오
- 영어 마케팅 용어를 남발하지 마십시오
- 경쟁사를 직접 비하하지 마십시오
- 과도한 이모지나 캐주얼한 톤을 사용하지 마십시오
- 자동화된 느낌의 대량 발송 메시지를 보내지 마십시오

## 1단계: 사전 개인화 리서치 (필수)

**아웃리치 메시지를 작성하기 전에 반드시 다음 정보를 수집하십시오.**

### 필수 수집 정보 체크리스트

| 항목 | 확인 여부 | 활용처 |
|------|-----------|--------|
| 대표이사/담당자 성함 및 정확한 직함 | 필수 | 호칭 |
| 기업의 최근 뉴스/성과 (3개월 이내) | 필수 | 오프닝 |
| 기업의 핵심 비즈니스/제품 | 필수 | 맥락 연결 |
| 추정되는 과제/니즈 | 필수 | 가치 제안 |
| 접촉 경로 (Warm Path) | 강력 권장 | 신뢰 구축 |
| 개인화 앵커 (인터뷰, 발표, SNS) | 강력 권장 | 관심 표현 |
| 유사 업종 성공 사례 | 권장 | 사회적 증거 |
| 정부 지원 사업 해당 여부 | 권장 | 예산 연결 |

이 정보가 COMPANY-RESEARCH.md 또는 DECISION-MAKERS.md에 있다면 참조하십시오. 없다면 sales-research 또는 sales-contacts 스킬을 먼저 실행할 것을 권고하십시오.

## 2단계: 채널 우선순위 결정

### 한국 B2B 아웃리치 채널 우선순위

| 순위 | 채널 | 응답률 (한국 SMB 추정) | 적합 상황 |
|------|------|------------------------|-----------|
| **1** | **소개/추천** | 40-60% | Warm Path가 있을 때 (최우선) |
| **2** | **전화** | 15-25% | 대표이사/핵심 담당자 직접 연락처 확보 시 |
| **3** | **카카오톡 비즈메시지** | 10-20% | 모바일 접근 (거의 모든 한국인이 사용) |
| **4** | **문자 (SMS/LMS)** | 8-15% | 짧은 팔로업, 부재중 후 |
| **5** | **이메일** | 3-8% | 상세 정보 전달, 자료 첨부 |

### 채널 선택 기준

```
Warm Path 있음? → YES → 1순위: 소개 요청 → 소개 후 전화
                  → NO  → 대표 직통번호 있음? → YES → 전화
                                                → NO  → 회사 대표번호로 전화 → 담당자 연결 시도
                                                        → 연결 실패 시 → 이메일 + 카카오톡
```

## 3단계: 7-Touch 멀티채널 시퀀스 (21일)

### 시퀀스 타임라인

| Touch | 일차 | 채널 | 목적 | 메시지 유형 |
|-------|------|------|------|-------------|
| **1** | Day 1 | 전화 (또는 소개) | 첫 접촉, 관계 형성 | 간결한 자기소개 + 가치 제안 |
| **2** | Day 1 | 이메일/카카오톡 | 전화 후속 또는 부재 시 메시지 | 소개 + 자료 링크 |
| **3** | Day 3 | 카카오톡/문자 | 가벼운 팔로업 | 전화 회신 요청 |
| **4** | Day 7 | 전화 | 재시도 | 새로운 가치 포인트 |
| **5** | Day 10 | 이메일 | 가치 컨텐츠 제공 | 사례/인사이트 공유 |
| **6** | Day 15 | 전화/카카오톡 | 관심 확인 | 간접적 확인 |
| **7** | Day 21 | 이메일 | 마지막 접촉 | 정리 + 문 열어두기 |

### Touch 1: 첫 전화 (Day 1)

#### 전화 스크립트 (30초 버전)

**시나리오 A: 대표이사 직접 연결**
```
안녕하십니까, {기업명} {대표이사 직함}님.
저는 {우리 회사}의 {이름}이라고 합니다.

{개인화 오프닝 - 아래 옵션 중 택 1}

혹시 잠깐 시간이 괜찮으시다면,
{기업명}에서 {추정 과제}와 관련하여
도움을 드릴 수 있는 부분이 있어 연락드렸습니다.

2-3분만 말씀 나눌 수 있을까요?
```

**개인화 오프닝 옵션:**
- (뉴스 기반) "최근 {뉴스 내용}을 보고 축하 말씀 드리고 싶어 연락드렸습니다."
- (소개 기반) "{소개자 이름} 대표님께서 한번 연락드려 보라고 말씀해 주셨습니다."
- (업종 기반) "저희가 {유사 업종}의 {고객사 예시}와 일하면서 {성과}를 달성한 경험이 있어, {기업명}에도 도움이 될 것 같아 연락드렸습니다."
- (이벤트 기반) "{행사/컨퍼런스}에서 {기업명}의 {발표/부스}를 인상 깊게 보았습니다."

**시나리오 B: 교환원/비서 연결**
```
안녕하세요, 저는 {우리 회사}의 {이름}이라고 합니다.
{대표이사명} 대표님과 통화가 가능할까요?
{IT 관련/업무 효율화 관련}으로 잠깐 말씀드릴 것이 있어서요.
```

**시나리오 C: 부재 시 (음성 메시지)**
```
{대표님/팀장님} 안녕하십니까.
{우리 회사} {이름}입니다.
{한 줄 가치 제안} 관련하여 연락드렸습니다.
{전화번호}로 회신 부탁드리겠습니다.
감사합니다.
```

### Touch 2: 첫 이메일 또는 카카오톡 (Day 1)

#### 이메일 템플릿

**제목 A/B 테스트 옵션:**
- A: `{기업명} {추정 과제} 관련 안내`
- B: `{대표이사 직함}님, {유사 기업} 사례 공유드립니다`
- C: `{소개자명} 대표님 소개로 연락드립니다`
- D: `{기업명}의 {목표}에 도움이 될 수 있는 방법`

**본문 (200자 내외):**
```
{대표이사 직함}님, 안녕하십니까.
{우리 회사} {이름}입니다.

{개인화 오프닝 1-2문장}

{기업명}에서 {추정 과제/니즈}를 검토하고 계실 것으로 생각되어
연락드렸습니다.

저희는 {유사 업종/기업}에서 {구체적 성과}를 달성한 경험이 있으며,
{기업명}에도 비슷한 도움을 드릴 수 있을 것으로 확신합니다.

짧은 시간이라도 말씀 나눌 수 있으시다면
편하신 시간을 알려주시겠습니까?

감사합니다.

{이름} 드림
{직함} | {우리 회사}
{전화번호} | {이메일}
```

#### 카카오톡 비즈메시지 (100자 내외)

## 4단계: 카피 작성 제약 조건

### 필수 준수 사항

#### 1. 존댓말 (경어체) 규칙
- 모든 메시지는 존댓말(합쇼체 또는 해요체)로 작성하십시오
- 대표이사: "대표님", "사장님"
- 임원: "{직함}님" (이사님, 전무님 등)
- 팀장: "팀장님", "{이름} 팀장님"
- 담당자: "{직함}님" 또는 "{이름}님"
- 절대 이름만 단독으로 부르지 마십시오 (예: "김철수" X → "김철수 대표님" O)

#### 2. 관계 구축 우선 (Relationship-First)
- 첫 메시지에서 바로 제품 판매를 시도하지 마십시오
- "도움을 드리고 싶다"는 자세를 유지하십시오
- 상대방의 성과나 노력을 먼저 인정하십시오
- 공감과 이해를 표현한 후 제안으로 이행하십시오

#### 3. 간접적 접근 (Indirect Approach)
- "사세요/구매하세요" 대신 "한번 살펴보시겠습니까?"
- "문제가 있으시죠?" 대신 "이런 부분에서 고민이 있으실 수도 있을 것 같습니다"
- "우리가 최고입니다" 대신 "유사 사례에서 이런 결과가 있었습니다"
- 단정적 표현 대신 여지를 남기는 표현을 사용하십시오

#### 4. 메시지 길이 제한
| 채널 | 최대 길이 | 이상적 길이 |
|------|-----------|-------------|
| 전화 스크립트 | 30초 (약 120자) | 20초 (약 80자) |
| 카카오톡 | 100자 | 60-80자 |
| 문자 SMS | 90바이트 (한글 45자) | 40자 |
| 문자 LMS | 2,000바이트 (한글 1,000자) | 200자 |
| 이메일 본문 | 300자 | 150-200자 |
| 이메일 제목 | 40자 | 20-30자 |

#### 5. A/B 테스트 제목 작성 규칙
이메일을 작성할 때 반드시 2개 이상의 제목 옵션을 제안하십시오:
- **옵션 A**: 기업명/개인화 요소 포함 (예: "{기업명} 업무 효율화 방안")
- **옵션 B**: 성과/숫자 포함 (예: "처리 시간 50% 단축한 방법")
- **옵션 C**: 소개/추천 기반 (해당 시) (예: "{소개자} 대표님 소개로 연락드립니다")

## 5단계: Outreach Readiness Score 산출 (0-100점)

### 평가 항목

| 항목 | 배점 | 설명 |
|------|------|------|
| 개인화 수준 (Personalization) | 0-25점 | 메시지가 얼마나 맞춤화되어 있는가 |
| 채널 최적화 (Channel Optimization) | 0-25점 | 적합한 채널을 선택했는가 |
| 메시지 품질 (Message Quality) | 0-25점 | 한국 비즈니스 예절에 맞고 설득력 있는가 |
| 시퀀스 완성도 (Sequence Completeness) | 0-25점 | 7-touch가 논리적으로 연결되어 있는가 |

#### 개인화 수준 (0-25점)
| 수준 | 점수 |
|------|------|
| 기업+인물+최근동향+개인앵커 모두 반영 | 21-25점 |
| 기업+인물+1-2개 개인화 요소 | 16-20점 |
